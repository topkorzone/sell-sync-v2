# PRD -- 오픈마켓 주문·배송·ERP 자동화 시스템

------------------------------------------------------------------------

## 1. 제품 상세 설명 (Detailed Product Description)

### 1.1 제품 개요

본 제품은 스마트스토어, 쿠팡, 11번가, 지마켓, 옥션 등 주요 오픈마켓의
주문 데이터를 통합 수집하여\
송장 발행, 택배 접수, ERP 자동 입력까지 전 과정을 자동화하는 B2B SaaS
시스템이다.

오픈마켓 운영자가 반복적으로 수행하던 주문 확인, 송장 입력, 택배 접수,
ERP 판매 입력 업무를 자동화함으로써\
인건비 절감, 휴먼에러 감소, 처리 속도 향상을 핵심 가치로 제공한다.

------------------------------------------------------------------------

### 1.2 문제 정의 (Problem Statement)

  기존 문제        상세 내용
  ---------------- --------------------------------------------
  주문 채널 분산   여러 오픈마켓 주문을 각각 확인해야 함
  반복 수작업      송장 발행, 택배 접수, ERP 입력을 수동 처리
  인건비 부담      주문량 증가 시 인력 증원이 불가피
  오류 발생        송장 번호 오입력, 누락, 정산 금액 불일치
  정산 관리 복잡   마켓별 수수료 정책이 달라 계산이 어려움

------------------------------------------------------------------------

### 1.3 해결 방안 (Solution)

-   주문 데이터를 30분 주기 자동 수집
-   주문 상태를 기준으로 자동 송장 발행 및 택배 접수
-   ERP 상품과 주문 상품을 사전 맵핑하여 자동 판매 등록
-   매일 새벽 1시 마켓 수수료를 반영한 판매 데이터 자동 입력

반복 업무 제거를 목표로 한다.
